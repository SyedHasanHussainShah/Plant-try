🧪 Testing Checklist

  1. Splash Screen: Verify the 3-second loading animation with growing plant
  2. Navigation: Test switching between Scan, History, and Community tabs
  3. Camera Access: Grant permissions and test photo capture (HTTPS required)
  4. File Upload: Test gallery selection and drag & drop functionality
  5. Analysis Progress: Watch the 3-step analysis process with progress bar
  6. Results Display: Check plant identification, diagnosis, and prescription
  7. History Management: Save results and view them in the History tab
  8. Responsive Design: Test on different screen sizes and orientations
  9. Toast Notifications: Verify success and error message displays
  10. Share Functionality: Test native sharing or clipboard fallback

🚀 Quick Start

Option 1: Python Server (Recommended)
python server.py --https
Then visit: https://localhost:8000
Option 2: Node.js Server
npm start
Then visit: http://localhost:8000
⚠️ Important: Camera functionality requires HTTPS. Use the --https flag with Python server or start:https with npm.

📱 Mobile Testing

For the best experience, test on a mobile device or use browser developer tools to simulate mobile view:

  • Open browser developer tools (F12)
  • Click the device toggle button
  • Select a mobile device (iPhone, Android, etc.)
  • Test touch interactions and responsive layout

🔧 Troubleshooting

Camera not working?
- Ensure you're using HTTPS
- Grant camera permissions when prompted
- Try the gallery upload as an alternative
Images not loading?
- Check browser console for errors
- Ensure all files are in the same directory
- Try refreshing the page
Server not starting?
- Check if port 8000 is available
- Try a different port: python server.py --port 8080
- Ensure Python 3.6+ is installed
🚀 Launch PlantDoctor
